# Analytics and Metrics

Raid Ledger tracks community engagement metrics to help admins understand participation patterns and identify trends.

## Available Metrics

### Attendance Tracking

- **Event attendance rates** — Percentage of signups who actually attended
- **No-show tracking** — Players who signed up but didn't attend
- **Signup trends** — How signup rates change over time

### Game Popularity

- **Events per game** — Which games have the most events
- **Player activity by game** — Which games are most actively played
- **Game time breakdown** — Aggregate time spent per game

### Community Engagement

- **Active players** — Number of players who participated in events
- **Event frequency** — How often events are created
- **Peak activity times** — When your community is most active

## User Profiles

Each user profile includes:
- **Events attended** — History of event participation
- **Game time** — Breakdown by game
- **Characters** — Linked game characters (via plugins)

## Admin Dashboard

The admin panel provides an overview of community health:
- Total events created
- Active users
- Signup rates
- Recent activity
